# pH-Wert & BGA — Lernwerkzeug

Interaktives Lernwerkzeug zur Autoprotolyse des Wassers, pH-Rechner und BGA-Simulation.

## Hosting auf GitHub Pages

1. Repository erstellen (z.B. `ph-bga`)
2. `index.html` hochladen
3. **Settings → Pages → Source**: `main` branch, `/ (root)`
4. Nach ~1 Minute erreichbar unter `https://<username>.github.io/ph-bga/`

## Lokal öffnen

```bash
open index.html
```

## Technologie

- Vanilla HTML/CSS/JS · Chart.js (CDN) · keine Build-Tools
- Dark/Light Mode automatisch · Responsive
